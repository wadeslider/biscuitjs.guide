# Instalando Node.js y Biscuit

Para usar Biscuit, tu necesitas instalar [Node.js](https://nodejs.org/), Biscuit v3 necesitas Node v18 en adelante.

```shell filename="Tip"
# Puedes ver tu version de node instalada con  el siguiente comando:

node -v

# Debe ser v18.x, si es asi puedes continuar leyendo.
```

Para instalar [Node.js](https://nodejs.org/) en windows, ve a la pagina oficial y descargue el instalador, lo ejecuta y listo.

Para macOS, puedes hacer lo siguiente:

- Descarga la ultima version de [Node.js](https://nodejs.org/), abre el instalador y sigue las instrucciones
- Usa el paquete [Homebrew](https://brew.sh/) con el comando ` brew install node `

En Linx consulta [esta pagina](https://nodejs.org/en/download/package-manager/) para determinar cómo debe de instalar Node.

## Preparando lo esencial

Para usar Biscuit, los necesitas instalar via npm (Node's package manager). npm se instala automaticamente cuando instalas node, asi que no hay que preocuparse por instalarlo. Sin embargo, antes de instalar algo debes configurar una carpeta para el proyecto.

## Abriendo la terminal

```shell filename="Tip"
# Si tu estas utilizando Visual Studio Code, debes presionar:

Ctrl + `

# (backtick)

# O

Ctrl + ñ

# para abrir la terminal integrado
```

Para Windows:

- ` Shift + Right-click ` dentro de tu proyectoy escoge "Abrir terminal"
- Presiona ` Win + R ` y ejecuta ` cmd.exe `, luego ` cd ` dentro de tu proyecto

Para macOs:

- Abra Launchpad o Spotlight y busque "Terminal"
- En su carpeta "Aplicaciones", en "Utilidades", abra la aplicación Terminal

Para linux: 

Debes presionar ` Ctrl + Alt + T `.

Con la terminal abierta, ejecuta ` node -v ` y si esta instalado devera devolver la version.

## Inicializar el proyecto

```shell npm2yarn
npm init
```

Este comando creara un archivo llamado ` package.json ` para ti, donde vendran las dependencias y otra informacion.

Este te pedira algunos datos y deberas ir respondiendo

Tambien puedes hacerlo automatico com:

```shell npm2yarn
npm init -y
```
