# Introducción

Esta guia es para los usuarios nuevos en Biscuit, esta es una libreria para hacer bots de discord, esta guia te va permitir hacer tu bot con ella, tambien otras cosas como:

- Como prender tu bot;
- Como crearlo y organizarlo;
- Como hacer embeds, botones, canvas;
- Como trabajar con algunas bases de datos;
- Como empezar con las shards;
- Y mucho mas;

Gracias a esta guia vas a poder tener tu codigo mas limpio y ordenado.

# Antes de empezar...

¡Perfecto! Hacer un bot es genial y todo, pero hay cosas que debes saber antes de continuar. Para empezar hacer bot con biscuit debes de tener un conocimiento decente de JavaScript o TypeScript. Aunque lo puedes lograr con muy pocos conocimiento de estos, intentar hacerlo sin saber alguno de estos lenguajes sera todo un reto para ti. Puedes que te quedes atascado ¡MUCHAS! veces (enserio, MUCHAS), pero recuerda que todo tiene una solucion.
